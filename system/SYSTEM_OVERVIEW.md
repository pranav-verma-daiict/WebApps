# Local Web Studio - System Overview

## Goal
Build and deliver professional websites for small local businesses in 2–3 hours using reusable templates and config-driven development.

## Core Principle
We do NOT build custom websites.
We fill pre-built systems with business data.

## Stack
- Next.js (App Router)
- Tailwind CSS
- Vercel deployment
- Static + config-driven rendering
- Formspree (contact forms)

## Constraints
- No custom design per client
- No new architecture per project
- All sites must use one of the templates
- Delivery time target: 2–3 hours

## Business Model
- One-time setup fee
- Optional monthly maintenance

## Templates
1. Local business (plumbers, cleaners, electricians)
2. Restaurant / café
3. Service businesses (salons, clinics, consultants)

## Success Definition
A website is successful if:
- It looks modern on mobile
- It clearly explains services
- It has a working contact method
- It loads fast
- It is deployed within 3 hours
