# Build Pipeline (2–3 hour system)

## Step 1: Intake (10–15 min)
Collect:
- Business name
- Services
- Phone/email
- Address
- Logo (optional)
- Images (optional)

---

## Step 2: Select template (5 min)
Choose:
- A (local business)
- B (restaurant)
- C (service business)

---

## Step 3: Clone project (5 min)

git clone starter-nextjs-app client-name

---

## Step 4: Fill config (20–30 min)

Edit:
business.config.ts

---

## Step 5: Content insertion (30–60 min)
- Replace text
- Replace images
- Adjust colors (optional)

---

## Step 6: Deploy (10 min)

vercel deploy

---

## Step 7: QA checklist (15 min)
- Mobile check
- Links working
- Form working
- Spelling
- Load speed

---

## Rule
No feature development during client work.
Only configuration.
